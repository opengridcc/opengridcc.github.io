Trim Zero's¶

Suppose you have a buch of zero's in your data, or you want to cut off data below a certain threshold

In [5]:
# turn every value below 100 into a 0

df[df < 100] = 0
In [6]:
_ = og.plotting.load_duration_curve(df, trim_zeros=True)
